## 1. Configuración Inicial <a id="configuración-inicial"></a>
Antes de cualquier medición, es imperativo asegurar la integridad geométrica del levantamiento:
* **Estacionamiento y Nivelación:** Centrado sobre el punto de control mediante plomada (óptica o láser) y nivelación del nivel tubular.
* **Orientación:** Establecimiento de línea base mediante acimut conocido o coordenadas de un segundo punto (Backsight).

## 2. Procedimientos Directos en Campo <a id="procedimientos-directos-en-campo"></a>

### Levantamiento por Radiación
Ideal para capturar detalles del terreno. Se recomienda el uso de códigos estandarizados para facilitar la importación en software CAD/GIS.

### Replanteo (Setting-Out)
Procedimiento inverso al levantamiento. La estación calcula la diferencia entre la posición actual del prisma y la coordenada teórica cargada en memoria.
* **S-O Line:** Para ejes constructivos.
* **S-O Arc:** Para trazo de curvas en infraestructura vial.

### Medición de Línea Faltante (MLM)
Utilizado para determinar distancias y desniveles entre puntos indirectos.

## Notas sobre Seguridad y Mantenimiento
* **Protección contra hongos:** Mantener el equipo en cajas secas con gel de sílice.
* **Verificación de Constantes:** Revisar la constante del prisma antes de iniciar la jornada para evitar errores sistemáticos de milímetros.
